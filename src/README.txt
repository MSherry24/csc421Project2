To compile: 
	From command line, navigate to ../src directory
	Use the following command: 

"javac code/ClosestPair.java code/Main.java code/MergeArray.java code/TextInputParser.java"

To run: Copy input files to ../src/code directory.
	Navigate to ../src directory.
	Use the following command:

java code/Main code/<inputs.txt>

	Where <inputs.txt> is your input file.
